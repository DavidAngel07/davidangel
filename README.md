# Portafolio

Este proyecto es un portafolio web moderno desarrollado con **React**, **TypeScript**, **Vite** y **Tailwind CSS**, diseñado para mostrar de forma profesional las habilidades, proyectos y experiencia como desarrollador web frontend.

---

## Tecnologías utilizadas

- **React** – Librería para construir interfaces dinámicas y modernas.
- **Vite** – Herramienta de desarrollo ultrarrápida para proyectos frontend.
- **TypeScript** – Tipado estático para un código más seguro y mantenible.
- **Tailwind CSS** – Framework CSS para estilos flexibles y personalizables.
- **shadcn/ui** – Componentes de interfaz reutilizables y estilizados.
